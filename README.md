# 🎬 Movie Ticket Booking System (Python)

## 📌 Overview

This is a simple **command-line based Movie Ticket Booking System** written in Python.
It allows users to:

- View available movies
- Select seat categories (Silver, Gold, Platinum)
- Book seats
- Calculate ticket prices
- Simulate payment and receive booking confirmation

---

## 🚀 Features

- 🎥 Display list of available movies
- 💺 Multiple seat categories with different pricing
- 🔢 Seat availability tracking
- 🧾 Automatic price calculation
- 💳 Simulated payment system
- ✅ Booking confirmation summary
- 🔁 Option to book multiple times

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open terminal/command prompt in the project folder.
4. Run the script:

```bash
python movie_recommendation_system.py
```

---

## 🎮 How It Works

1. The system displays available movies.
2. User selects a movie (1–3).
3. Seat categories and prices are shown.
4. User selects a category (Silver/Gold/Platinum).
5. Available seats are displayed.
6. User enters number of tickets.
7. User selects seats one by one.
8. Total price is calculated.
9. Payment is simulated.
10. Booking confirmation is displayed.
